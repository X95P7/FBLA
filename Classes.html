What we do

Our classes are made to teach the rising generation about our everchanging technological world. Through our unique teaching methods, students will be able to grasp, grow, and master the art of programming as well as expand their problem-solving abilities. With a variety of classes, we are able to teach people of all stages of learning, creating a system in which one can expand their knowledge with ease.

Walking

Description

In this class, we teach people the basics of programming using block code. We will start the class by introducing the abilities of programming through several examples. Afterward will begin the major portion of the class where the students will be introduced to block-based programming and simple conventions such as the for loop and the if-then logic block. By the end of the class, the students will have an understanding of the fundamentals of programming and will create either a small game or animation.

Running

Description

In the class, we introduce the concept of written code to our students. This will be done by creating programs in a block-coding environment and then recreating them in written langue. This practice allows for our students to draw parallels between the concepts much more easily and to have an intuitive reference base. By the end of this class, students will be able to understand how written code functions and the ability to identify relations between written and block coding.

Soaring

Description

This class focuses more on the logical side of programming and the creation of algorithms. During this class, students will use problem-solving skills and the java coding language to create solutions. Instructors will lead the class for the first part of the class demonstrating the basic components used in creating algorithms such as variables, for loops, and if-then statements along with pseudo-code to ensure comprehension. The latter half of the class will be given to the students to try to solve problems with their groups. By the end of the class, students will have a strong understanding of algorithm structures and logical problem-solving.
