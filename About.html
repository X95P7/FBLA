Overview

We offer 4 levels of classes designed to help based on both their level of programming skills and their age though anyone can take any class they wish. Each class is 2-3 hours long, during this time you will be taught by 3-4 qualified individuals helping you each set of the way. The class environment is designed to help you be comfortable and collaborative with the other people in the room, allowing you to learn and socialize with others. If you're interested in joining a class you can do so by filling out the forum on our Contact page.

Our Classes

Our Walking class is designed for children between the ages of 6-12 and utilizes block code to help teach children the fundamentals of programming logic.

The Running class is designed for children aged 10-14, this class begins to introduce written code along with block code to help show the connection between the two.

Our Climbing class is recommended for all ages 13+ and is the first class to be exclusively written code. In this class, we teach the basics of website design and use javascript to interact with the page!

The Soaring class is also recommended for ages 13+ with an interest in the more logical side of programming. The class will teach you to create algorithms to solve problems in a group setting.

Benefits of our classes

Groups

All of our classes are within a group environment, creating opportunities for collaboration and interaction. You will be able to work with the people around you to solve problems, making for a more productive and enjoyable lesson.

Environment

Our classes offer high-quality computers, chairs, and desks to ensure that you are in a comfortable environment. Air-conditioning and heating are supplied to allow for classes to run year-round as well. Along with our seating arrangement, designed to maximize interaction between students, our classes create a perfect environment for growth.

Instructors

Each class has multiple instructors to make sure each student feels welcomed in our classes. All certified and well-practiced, they will be able to help students 1 on 1 to ensure all students master the lessons taught in a stress-free manner.

Opportunity for growth

Our classes are designed to help our students grow their problem-solving abilities and to teach them the art of programming. Each class builds on the other and offers resources to those interested in learning outside of class. This formula allows for students to grow their skills over time, even if they only take one class.
